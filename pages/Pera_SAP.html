Pera SAP-I

This is an 8-bit Minimum Computer (Computation Machine) that made from simple logic ICs, flip-flops and a few memory modules. It has a memory of 16 bytes and its maximum clock speed is not more than 5 Hz. The whole design is based on SAP-I Architecture and design was obtained from Ben Eater’s Youtube Channel. Currently, it has 9 instructions (based on Intel 8080 instructions) and can be programmed it using machine language (That means programming from 1s and 0s)
